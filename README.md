# Plymouth Gangster Edition
A Plymouth theme made from meme

![Theme demo](animation/anim-28.png)

## Manual installation

Clone the repo to `/usr/share/plymouth/themes` and run `# plymouth-set-default-theme -R plymouth-gangster-edition`
